Project Idea

This is website created for the helping of the farmer to sell their products directly to the wholesaler dealers in big cites.

Technology Used in the Project

  • Django
  • Sqlite3 Database
  • Bootstrap
  • javascript

Language Used in the project

  • Python
  • HTML
  • Javascript

Features Available in this demo website

Farmers

  • Farmer can register himself/herself by providing the neccessary informartion like:-
  • Village, Name,Age
  • Farmer can add their multile crops like:-
  • Wheat, Bajra, Rice etc
  • Farmer can connect to the outside world to the distributer directly.

Buyers

  • Buyers can signup in the website and create an account-
  • Buyers can use the facility of cart after creating account
  • Buyers can place order whatever they need as per requirement
